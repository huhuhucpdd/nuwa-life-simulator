## 超能力一览

- **时间线编辑器**  
  不是被动测试，是主动博弈。你可以随时说：“回到 2023 年 6 月，我拒绝那份海外工作，如果我当时接受会怎样？”

- **自我蒸馏引擎**  
  导入你的聊天记录、社交媒体数据或直接对话，系统会在隐私沙盒中提炼你的行为模式——而且会自动抵抗自我美化偏差。

- **量子人生模拟器**  
  基于你的真实特征与大量匿名路径原型，生成 α/β/γ 三条时间线，并用混合引擎（统计模型 + 个人偏移 + 随机）推演未来。

- **事件牌库**  
  随时触发意外事件：“行业黑天鹅”“贵人出现”……为人生推演加点戏剧性。

- **平行宇宙对比卡**  
  一句命令对比不同时间线上 5 年后的自己：收入、情绪、自由感，一眼看穿。

- **100% 题目跳过权**  
  题库很丰富，但你可以跳过任一道题、任一类题，甚至所有题，直接以默认特征开演。
