## 능력 요약

- **타임라인 편집기**  
  수동적 테스트가 아닌 능동적 게임. “2023년 6월로 돌아가서 해외 취업 제안을 거절했을 때, 수락했으면 어땠을까?”라고 말하세요.

- **자기 증류 엔진**  
  채팅 기록이나 SNS 데이터를 입력하면 프라이버시 샌드박스 안에서 행동 패턴을 추출하고, 자기 미화 편향을 자동 보정합니다.

- **양자 인생 시뮬레이터**  
  당신의 실제 특성과 대규모 익명 경로 데이터베이스를 기반으로 α/β/γ 세 개의 타임라인을 하이브리드 엔진으로 생성합니다.

- **이벤트 덱**  
  “업계 블랙 스완”, “멘토 등장” 등 무작위 사건을 주입해 극적인 반전을 더합니다.

- **평행 우주 카드**  
  서로 다른 타임라인의 5년 후 수입, 감정, 자유도를 한눈에 비교.

- **질문 완전 생략 가능**  
  방대한 질문 은행이 있지만, 모든 질문, 질문 유형, 전체 질문을 건너뛰고 기본 특성으로 바로 실행할 수 있습니다.
