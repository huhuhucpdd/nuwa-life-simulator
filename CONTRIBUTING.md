# 贡献指南

感谢你有兴趣为赛博女娲项目做出贡献！

## 如何贡献

1. **Fork 本仓库**。
2. 创建你的特性分支 (`git checkout -b feature/你的特性`)。
3. 提交你的修改 (`git commit -m '添加某个特性'`)。
4. 推送到分支 (`git push origin feature/你的特性`)。
5. 创建一个 Pull Request。

## 参与方向

- 丰富 `references/question-bank.md` 中的题库（请保持题型分类和反美化探测器）。
- 扩展 `references/event-deck.md` 中的事件卡牌。
- 为不同语言提供本地化版本 README 或示例会话。
- 改进推演引擎的叙述模板。
- 报告 Bug 或提出增强建议，通过 Issues 板块。

## 交流讨论

欢迎通过 Issues 或邮件联系作者进行讨论。你的每一个想法都可能让这个时间编辑室更有乐趣。
